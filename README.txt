
برنامج إدارة قروب كرة القدم — شرح الاستخدام الكامل

1- صفحة اللاعبين
- إضافة لاعب جديد.
- حذف لاعب.
- تظهر جميع الأسماء بخط موحد.

2- صفحة اللعبة
- اختيار تاريخ اللعبة.
- اختيار المشاركين.
- تحديد سعر الملعب.
- تقسيم الفرق.
- حفظ اللعبة.

3- صفحة الفريقين
- اختيار اللعبة المحفوظة.
- عرض الفريق الأول والثاني.
- عرض المشاركين أسفل الصفحة مثل صفحة التقويم.

4- صفحة الإيداعات
- اختيار اللاعب.
- إدخال مبلغ.
- اختيار نوع العملية:
  • إيداع
  • خصم / مديونية
  • تأخير
- جميع العمليات تظهر بالأسفل بالأحدث أولاً.
- يشمل ذلك الإيداع المبدئي.

5- صفحة الجدول
- عرض ترتيب اللاعبين.
- الضغط على اسم اللاعب يفتح كشف اللاعب تلقائياً لنفس الاسم.

6- صفحة التقارير
- عرض الرصيد وعدد اللعب والإيداعات.
- الضغط على اسم اللاعب يفتح كشف اللاعب تلقائياً.

7- صفحة كشف اللاعب
- عرض:
  • الرصيد
  • أيام اللعب
  • الإيداعات والمديونيات
  • الضيوف الذين أحضرهم اللاعب
- ترتيب العمليات بالأحدث أولاً.

8- صفحة التقويم
- عرض الألعاب حسب التاريخ.
- عرض المشاركين والفرق.

9- النسخ الاحتياطي
- تصدير نسخة احتياطية.
- استيراد نسخة احتياطية.

متطلبات الاستخدام
- يفضل فتح البرنامج عبر Safari أو Chrome.
- بعد أي تحديث:
  1- احذف الموقع القديم من الشاشة الرئيسية.
  2- افتح الرابط الجديد.
  3- حدث الصفحة.
- عند ظهور نسخة قديمة:
  • امسح Cache المتصفح.
