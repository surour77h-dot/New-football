نسخة Apple Sports Ultra Safe

هذه نسخة آمنة:
- لم يتم حذف أو إخفاء الصفحات الأصلية.
- تم تغيير الهيدر والبطاقات والخلفيات والأزرار والجداول.
- تم إضافة Dashboard وشريط تنقل سفلي.
- تم الحفاظ على شريط الصفحات الأصلي أيضاً حتى لا تظهر صفحة فارغة.
- تم تحديث Service Worker لكسر الكاش.

إذا ظهر الشكل القديم:
احذف اختصار التطبيق من الشاشة، أو افتح الرابط في نافذة خاصة، أو امسح بيانات الموقع.
