هذه نسخة إعادة تصميم كاملة: Apple Sports Ultra

التغيير ليس ثيم فقط:
- Dashboard رئيسية جديدة.

- بطاقات إحصائية.
- تغيير شكل البطاقات والجداول والقوائم والأزرار.
- الحفاظ على وظائف التطبيق والبيانات.
