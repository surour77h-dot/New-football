قروب الكورة - New App

هذه ليست نسخة ثيم.
تم بناء هيكل تطبيق جديد:
- App Shell جديد.
- Header جديد.
- Dashboard رئيسية.
- Side Navigation للشاشات الكبيرة.
- Bottom Navigation للآيفون.
- ترتيب صفحات جديد.
- بطاقات وأزرار وجداول جديدة.

تم الحفاظ على:
- نفس app.js الأساسي.
- نفس أسماء الحقول والصفحات.
- نفس التخزين المحلي localStorage.
- نفس الحسابات والمعادلات قدر الإمكان.

ملاحظة:
إذا ظهرت نسخة قديمة بعد الرفع، احذف اختصار التطبيق القديم أو امسح بيانات الموقع بسبب كاش PWA.
