دليل استخدام تطبيق قروب الكورة

الملفات الأساسية:
1- index.html
2- style.css
3- app.js
4- manifest.json
5- sw.js

طريقة التشغيل:
- ضع الملفات في مجلد واحد.
- افتح index.html أو ارفعها على Netlify أو GitHub Pages.
- يمكن تثبيت التطبيق كتطبيق PWA على iPhone و iPad.

الصفحات:
- الرئيسية: إنشاء وتعديل المباريات.
- اللاعبين: إدارة اللاعبين.
- جدول اللاعبين: الأرصدة والإحصائيات.
- كشف لاعب: تفاصيل كل لاعب.
- التقويم: عرض المباريات حسب التاريخ.
- تقسيم الفريقين: توزيع اللاعبين على فريقين.
- الإيداعات: تسجيل الإيداعات والمديونيات والتأخير.
- الحسابات: ملخص المبالغ والخصومات والإضافات.
- سجل المباريات: جميع المباريات السابقة.
- الإعدادات: النسخ الاحتياطي وتخصيص التطبيق.

ملاحظات:
- جميع البيانات تحفظ محلياً داخل المتصفح.
- يفضل أخذ نسخة احتياطية دورية.
- بعد أي تحديث كبير يفضل تحديث كاش التطبيق.
